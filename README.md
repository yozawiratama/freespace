###Description
fSpace or freeSpace is a template engine for meteor to create a Social Network.


###Features
 - Status Update
 - Register
 - Login

###Requirements
Meteor framework installed.

###Setup
Use your terminal. Just enter to fSpace folder and say "meteor".
